# Разработка программных модулей - 5 семестр

## О курсе
Практический курс по разработке backend-приложений на Python с использованием FastAPI и других современных технологий.

| 1 | Повторение Python | Освежить знания о синтаксисе и ключевых конструкциях языка Python, модулям и ООП, подготовить почву для углубленной backend-разработки Python |
| 2 | Архитектура проектов | Познакомиться с основами проектирования приложений: структурой каталогов, слоями ответственности и практиками организации кода. Научиться создавать виртуальное окружение и сохранять зависимости проекта в файле requirements.txt. |
